Built for your sport

Every sport gets
its own features.

Pickup isn't generic — each sport has fields, filters, and formats that make sense for how you actually play.

Pickleball
  • DUPR rating input
  • Game vibe: Social / Competitive / Bangers OK
  • Kitchen rules enforced toggle
  • Round-robin format support
Tennis
  • NTRP minimum rating filter
  • Court surface: Hard / Clay / Grass
  • Match type: Singles / Doubles / Rally
  • NTRP auto-tagged in post
Basketball
  • Format: 3v3 / 4v4 / 5v5 / 21 / Open
  • Auto-sets roster by format
  • House rules text field
  • Ability level filter
Running
  • Pace range (min/max per mile)
  • Run type: Easy / Tempo / Trail / Social
  • Distance + elevation gain
  • Route URL link

How it works

From install to game
in under 2 minutes.

1
Download and pick your sports
Choose the sports you play. Pickup immediately shows you every open game nearby, sorted by distance and start time. Set your skill level once — games filter to match automatically.
2
Browse the live map
See a pin for every game near you. Tap any pin for full details — sport, host, skill level, who's already in, and spots remaining.
3
Join a game or post your own
Tap Join and you're in — spots update in real time. Or tap + to post your own game, fill the sport-specific fields, and watch your spots fill.
4
Show up, play, and log the result
Get a push notification before tip-off. Check in at the court. After the game, post the score — it counts toward your record and the local leaderboard.
