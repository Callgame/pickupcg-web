Built for your sport

Every sport gets
its own features.

Pickup isn't a generic game finder — each sport has fields, filters, and formats that actually make sense for how you play.

🏓 Pickleball
  • DUPR rating input
  • Game vibe: Social / Competitive / Bangers OK
  • Kitchen rules enforced toggle
  • Round-robin format support
🎾 Tennis
  • NTRP minimum rating filter
  • Court surface chips (Hard / Clay / Grass)
  • Match type: Singles / Doubles / Rally
  • NTRP auto-tagged in game post
🏀 Basketball
  • Format chips: 3v3 / 4v4 / 5v5 / 21
  • Auto-sets roster size by format
  • House rules text field
  • Open run option
🏃 Running
  • Pace range (min/max per mile)
  • Run type: Easy / Tempo / Trail / Social
  • Distance and elevation gain
  • Route URL field

How it works

From install to game
in under 2 minutes.

1
Download and pick your sports
Choose the sports you play. Pickup immediately shows you every open game nearby, sorted by distance and start time. Set your skill level once — games filter to match you automatically.
2
Browse the live map
See a pin for every game happening near you. Tap any pin for details — sport, host, skill level, who's already joined, and how many spots are left.
3
Join a game or post your own
Tap Join and you're in — spots decrement in real time. Or tap + to post your own game. Fill the sport-specific fields, set your time and location, and watch your spots fill.
4
Show up, play, and log the result
Get a push notification before tip-off. Check in at the court to connect with other players there. After the game, post the score — it counts toward your record and the local leaderboard.
